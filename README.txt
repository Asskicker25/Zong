INSTRUCTIONS

1. You can find the unity project inside Zong_Test folder at the root. 
2. All the necessary files for the project is under Assets/ZongTest folder.
3. You can find the build files inside the Build Files folder at the root.
4. Open the Zong_Test.exe to play the game.

PLAYER CONTROLS

1. Move						- WSAD and mouse
2. Pick up item				- E/F
3. Drop at box				- E/F
4. Open Inventory anytime - SPACE
